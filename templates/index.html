Jokes API

This is an API for Chuck Norris facts/jokes. It is a API proxy for api.chucknorris.io.
The original API is supplemented by a new (fake) attribute author. This information is stored persistent. The first time a joke is fetched, it gets assigned a random author, which is then saved in a PostgreSQL database.

Endpoints:

Get random joke GET https://jokes-api100.herokuapp.com/joke
Get random joke with reversed words GET https://jokes-api100.herokuapp.com/joke/reverse
Get joke by id GET https://jokes-api100.herokuapp.com/joke/{id}
Get joke by id with reversed words GET https://jokes-api100.herokuapp.com/joke/{id}/reverse
Add new author POST https://jokes-api100.herokuapp.com/author
Get author by id GET https://jokes-api100.herokuapp.com/author/{id}
Get all authors GET https://jokes-api100.herokuapp.com/author
Update author by id PUT https://jokes-api100.herokuapp.com/author/{id}
Delete author by id DELETE https://jokes-api100.herokuapp.com/author/{id}
Assign author to joke PUT https://jokes-api100.herokuapp.com/joke/{jid}/{aid}
